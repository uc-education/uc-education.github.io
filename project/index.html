"2+2"大学联读

  • 内容概览
  • 背景知识
  • 适合人群
  • 留学内容
  • 费用估算
  • 何谓2+2？

    首先入读两年制的社区学院，毕业后转学进入四年制综合大学的大三，两年后本科毕业获得学士学位。因为是在社区学院和综合大学各读了两年，因此称为“2+2”。

  • 入学条件：

    高中毕业生，或在读大学生。外国留学生无需SAT考试成绩，无需TOEFL英语成绩。

  • 留学时间：

    社区学院2年+大学2年，总时长为4年。但中国学生一般都需要增加一年的英语提高课程，则社区学院3年+大学2年，总时长为5年。

  • 留学费用：

    社区学院期间为2-2.5万美元（包括学费和食宿生活费用）。大学期间为5-6万美元（包括学费和食宿生活费用）。

  • 入学时间：

    每年春/夏/秋冬季入学，提前两个月进行申请。

  • 特色优势：

    这是美国特有的上大学方式，入学门槛低、上学费用低。尤其是，每所社区学院都与许多大学签有直接转学协议，其中包括为数众多的世界名校。比如雪兰学院与22所大学签有直接转学协议，其中就包括著名的华盛顿大学（全球大学排名第13），在雪兰学院获得相应的学分和GPA成绩，即可直接转升到华盛顿大学。

      • 对于我们国内的学生来说，如果想上大学，就必须首先在高中获得学籍，然后参加全国统一高考并竞争胜出，并且在填报志愿时一旦选定了学校和专业，就必须是一条路走到底直到毕业，中间几乎不存在转换调整的可能性。

      但美国则完全是另外一番景象：申请大学的方式自由而开放，并且上大学的方式也是多种多样的。并且在大学期间，学生不仅可在校内自由转换系科专业，而且还可以在全国大学间自由转学。正是这种开放的教育环境和完善的制度保障，将美国的大学打造成了全球最高水平的高等教育系统。在全球TOP100名校中有51所美国大学，而在TOP20顶级名校中，美国大学更是占据了16席。

      美国有近4000所大学，其中1/3是两年制社区学院，其他则是四年制的综合大学，转学可在各种不同类型学校间进行，全美高校每年有超过45%的学生进行校际间的转学。前总统奥巴马和现任总统特朗普都是从地方学院转学进入常青藤名校的典型例子。

      社区学院是美国公立大学系统的重要部分，主要为本地居民提供初级的高等教育课程和职业教育培训，费用只有一般大学的20-30%，并且没有入学条件限制，因此是最受普通家庭欢迎的大学类型，每年美国有近半数的高中毕业生进入本地的社区学院，两年毕业后再转升进入综合大学继续深造。

    如果你是：

    1、高考失利的高中毕业生；

    2、没有高中毕业证（如3+2模式或五年制大专）的在读生或毕业生；

    3、国内大学在读生或毕业生（大一大二学生最好）；

    4、年满16周岁的国内初中毕业生。

    如果你想留学美国，那么这条路就是最适合你的。

    但同时我们也必须看到，虽然这是一条开放而人性的阳光大道，但是机遇从来都是给有着强烈成功愿望的人而准备的，世上绝不会有不经努力而唾手可得的所谓“好机会”。

    因此，这个“2+2大学联读”项目对加入者还是有明确选择性的，如果你符合如下条件，请联系我们，我们一定会助你成功！

    1、有强烈的通过自身努力而创造美好未来的渴望；

    2、对留学美国有基本的了解和认识，希望能通过美国留学使自己获得更大的提高，为未来发展打下良好基础；

    3、自我管理能力强，学习能动性强；

    4、有独立自理的生活能力；

    5、家庭在留学支出上有预算。

    "16+"高中大学联读

    • 内容概览
    • 背景知识
    • 适合人群
    • 留学内容
    • 费用估算
  • 何谓16+？

    这是美国华盛顿州特有的教育政策，年满16周岁的学生，首先入读两年制的社区学院，毕业后同时获得高中毕业证和学院毕业证，然后转学进入四年制综合大学入读大三，两年后本科毕业获得学士学位。

  • 入学条件：

    入学时年满16周岁（相当于国内初三以上），或在读高中生。外国留学生无需SAT考试成绩，无需TOEFL英语成绩。

  • 留学时间：

    社区学院2年+大学2年，总时长为4年。但中国学生一般都需要增加一年的英语提高课程，则社区学院3年+大学2年，总时长为5年。

  • 留学费用：

    社区学院期间为2-2.5万美元（包括学费和食宿生活费用）。大学期间为5-6万美元（包括学费和食宿生活费用）。

  • 入学时间：

    每年春/夏/秋/冬季入学，提前两个月进行申请。

  • 特色优势：

    这个项目相当于是“2+2大学联读”的低龄化版本，即在美国的社区学院同时完成高中和大学一二年级的学业。尤其对于那些不打算在国内上高中，或是没有学籍无法参加高考的学生来说，这是最具优势的留学项目。

    • 项目起源

    1990年3月，在华盛顿州政府和比尔与梅琳达盖茨基金会的支持下，华盛顿州立法机构在 “Learning By Choice”教育法案中RCW28A.600.400设立了高中大学连读课程项目（High School Completion program）项目。

    该项目旨在培养美国11和12年级（即高三和高四年级）优秀高中生（GPA在B+以上），由州政府指定学校为其提供浓缩的高中及大学联读课程，使优秀学生能够缩短学习时间，提前完成高中课程，获得高中毕业证并修读大学课程。 优秀的美国高中学生可以跳过高中一到两年的课程，直接进入社区学院“大学预备班”，经过两年的学习，同时获得华盛顿州高中文凭和两年制学院的大学专科学历，然后升入理想的大学继续读大学三四年级，并获取大学本科学位。 由此，学生可以节省一到两年的时间获取学士学位。

    近年来，华盛顿州将该项目对国际学生开放，具备良好的英语能力的国际学生可直接进入此项目，如果英语能力不足，则可首先进入社区学院完成英语强化学习，然后再进入该项目直接学习专业课程。

    项目特色：

    　　

    节约学习时间：从高中到本科毕业只需要4-5年。经典的路线是，16岁入学社区学院，20或21岁从华盛顿大学本科毕业。 　　

    节约学习费用：比传统升学路线节省近60万元（节省了高中留学费用）。 　　

    入学要求较低：无需语言成绩亦可录取。首先由社区学院录取，一般进行一年的英语强化提高学习。 　　

    名校保障录取：入学即可同时获得美国名校条件录取。社区学院与华盛顿州的大学都有转升协议。
